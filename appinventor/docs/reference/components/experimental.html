FirebaseDB

Non-visible component that communicates with Firebase to store and retrieve information.

Additional Information

Properties

DeveloperBucket (designer only)
FirebaseToken (designer only)
FirebaseURL
Gets the URL for this FirebaseDB.
Persist (designer only)
If true, variables will retain their values when off-line and the App exits. Values will be uploaded to Firebase the next time the App is run while connected to the network. This is useful for applications which will gather data while not connected to the network. Note: AppendValue and RemoveFirst will not work correctly when off-line, they require a network connection.

Note: If you set Persist on any Firebase component, on any screen, it makes all Firebase components on all screens persistent. This is a limitation of the low level Firebase library. Also be aware that if you want to set persist to true, you should do so before connecting the Companion for incremental development.
ProjectBucket
Gets the ProjectBucket for this FirebaseDB.

Events

DataChanged(text tag, any value)
Indicates that the data in the Firebase has changed. Launches an event with the tag and value that have been updated.
FirebaseError(text message)
Indicates that the communication with the Firebase signaled an error.
FirstRemoved(any value)
Event triggered by the "RemoveFirst" function. The argument "value" is the object that was the first in the list, and which is now removed.
GotValue(text tag, any value)
Indicates that a GetValue request has succeeded.
TagList(list value)
Event triggered when we have received the list of known tags. Used with the "GetTagList" Function.

Methods

AppendValue(text tag, any valueToAdd)
Append a value to the end of a list atomically. If two devices use this function simultaneously, both will be appended and no data lost.
GetTagList()
Get the list of tags for this application. When complete a "TagList" event will be triggered with the list of known tags.
GetValue(text tag, any valueIfTagNotThere)
GetValue asks Firebase to get the value stored under the given tag. It will pass valueIfTagNotThere to GotValue if there is no value stored under the tag.
RemoveFirst(text tag)
Return the first element of a list and atomically remove it. If two devices use this function simultaneously, one will get the first element and the the other will get the second element, or an error if there is no available element. When the element is available, the "FirstRemoved" event will be triggered.
ClearTag(text tag)
Remove the tag from Firebase
StoreValue(text tag, any valueToStore)
Asks Firebase to store the given value under the given tag.
If you are having difficulty with the Companion and you are switching between different Firebase accounts, you may need to use this function to clear internal Firebase caches. You can just use the "Do It" function on this block in the blocks editor. Note: You should not normally need to use this block as part of an application.

BluetoothLE

This is a trial version of BluetoothLE component, blocks need to be specified later

id="BluetoothLE">Properties

AdvertisementScanPeriod
Returns the value of ScanPeriod.
BatteryValue
Return the battery level.
ConnectedDeviceRssi
Return the RSSI (Received Signal Strength Indicator) of connected device.
DeviceList
Return a sorted list of BluetoothLE devices as a String.
GetAdvertiserAddresses
Returns a list of the addresses of devices found during Advertisement scanning.
GetAdvertiserNames
Returns a list of the names of the devices found during Advertisment scanning.
IsDeviceAdvertising
Returns true if the device is currently advertising, false otherwise.
IsDeviceConnected
Return true if a BluetoothLE device is connected; Otherwise, return false.
TxPower
Return the Tx power.

Events

ByteValueChanged(text byteValue)
Trigger event when byte value from connected BluetoothLE device is changed.
ByteValueRead(text byteValue)
Trigger event when byte value from connected BluetoothLE device is read.
Connected()
Trigger event when a BluetoothLE device is connected.
DeviceFound()
Trigger event when a new BluetoothLE device is found.
FloatValueChanged(number floatValue)
Trigger event when int value from connected BluetoothLE device is changed.
FloatValueRead(number floatValue)
Trigger event when byte value from connected BluetoothLE device is read.
IntValueChanged(number intValue)
Trigger event when int value from connected BluetoothLE device is changed.
IntValueRead(number intValue)
Trigger event when int value from connected BluetoothLE device is read.
RssiChanged(number device_rssi)
Trigger event when RSSI (Received Signal Strength Indicator) of found BluetoothLE device changes
StringValueChanged(text stringValue)
Trigger event when String value from connected BluetoothLE device is changed.
StringValueRead(text stringValue)
Trigger event when String value from connected BluetoothLE device is read.
ValueWrite()
Trigger event when value is successfully written to connected BluetoothLE device.

Methods

Connect(number index)
Connect to a BluetoothLE device with index. Index specifies the position in BluetoothLE device list, starting from 0.
ConnectWithAddress(text address)
Connect to BluetoothLE device with address. Address specifies bluetooth address of the BluetoothLE device.
DisconnectWithAddress(text address)
Disconnect from connected BluetoothLE device with address. Address specifies bluetooth address of the BluetoothLE device.
text FoundDeviceAddress(number index)
Get the address of found device with index. Index specifies the position in BluetoothLE device list, starting from 0.
text FoundDeviceName(number index)
Get the name of found device with index. Index specifies the position in BluetoothLE device list, starting from 0.
number FoundDeviceRssi(number index)
Get the RSSI (Received Signal Strength Indicator) of found device with index. Index specifies the position in BluetoothLE device list, starting from 0.
text GetAdvertisementData(text deviceAddress, text serviceUuid)
Returns the advertisement data associated with the specified device address.
text GetAdvertiserAddress(text deviceName)
Returns the address of the device with the name specified.
list GetAdvertiserServiceUuids(text deviceAddress)
Returns the list of services available on the Adverising device.
text GetCharacteristicbyIndex(number index)
Return Unique ID of selected characteristic with index. Index specified by list of supported characteristics for a connected device, starting from 0.
text GetServicebyIndex(number index)
Return Unique ID of selected service with index. Index specified by list of supported services for a connected device, starting from 0.
text GetSupportedCharacteristics()
Return list of supported characteristics for connected device as a String
text GetSupportedServices()
Return list of supported services for connected device as a String
ReadByteValue(text service_uuid, text characteristic_uuid)
Read Byte value from a connected BluetoothLE device. Service Unique ID and Characteristic Unique ID are required.
ReadFloatValue(text service_uuid, text characteristic_uuid, number floatOffset)
Read Float value from a connected BluetoothLE device. Service Unique ID, Characteristic Unique ID and offset are required. Offset specifies the start position of reading data.
ReadIntValue(text service_uuid, text characteristic_uuid, number intOffset)
Read Integer value from a connected BluetoothLE device. Service Unique ID, Characteristic Unique ID and offset are required. Offset specifies the start position of reading data.
ReadStringValue(text service_uuid, text characteristic_uuid, number strOffset)
Read String value from a connected BluetoothLE device. Service Unique ID, Characteristic Unique ID and offset are required. Offset specifies the start position of reading data.
ScanAdvertisements(number scanPeriod)
Scans for Bluetooth LE advertisements. scanPeriod specifies how long the scan will run.
StartAdvertising(text inData, text serviceUuid)
Create and publish a Bluetooth LE advertisement. inData specifies the data that will be included in the advertisement. serviceUuid specifies the UUID of the advertisement.
StartScanning()
Start Scanning for BluetoothLE devices.
StopAdvertising()
Stop Bluetooth LE Advertising.
StopScanning()
Stop Scanning for BluetoothLE devices.
StopScanningAdvertisements()
Stops scanning for Bluetooth LE advertisements.
WriteByteValue(text service_uuid, text characteristic_uuid, text value)
Write byte value to a connected BluetoothLE device. Service Unique ID, Characteristic Unique ID, Integer value and offset are required. Offset specifies the start position of writing data.
WriteFloatValue(text service_uuid, text characteristic_uuid, number value, number offset)
Write Float value to a connected BluetoothLE device. Service Unique ID, Characteristic Unique ID, Integer value and offset are required. Offset specifies the start position of writing data.
WriteIntValue(text service_uuid, text characteristic_uuid, number value, number offset)
Write Integer value to a connected BluetoothLE device. Service Unique ID, Characteristic Unique ID, Integer value and offset are required. Offset specifies the start position of writing data.
WriteStringValue(text service_uuid, text characteristic_uuid, text value)
Write String value to a connected BluetoothLE device. Service Unique ID, Characteristic Unique ID and String valueare required.
